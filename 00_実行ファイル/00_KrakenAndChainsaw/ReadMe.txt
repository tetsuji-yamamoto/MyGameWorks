◇制作人数 : 一人
◇作品名   : KrakenAndChainsaw
◇使用言語 : C++
◇SDK      : DirectX 9
◇開発期間 : 2025/11 ～ 2026/01（約3か月）
◇ジャンル : 3Dアクションゲーム

◆操作方法
移動         : [W] [A] [S] [D]
ジャンプ     : [SPACE]
攻撃(長押し) : [F]

◆特徴
本作品は技術的チャレンジを盛り込んだ作品となっております。
エディター制作での反省から、内部構造にコンポーネント指向を取り入れて汎用性と拡張性を上げました。、
スキンメッシュを取り入れたクラーケンの表現やメッシュでの当たり判定などに力を入れて制作しました。
HLSLを使った描画にも挑戦しています。

◆反省
技術的にチャレンジしすぎてしまいゲーム本編の制作が押してしまいました。
スケジュール管理とスキンメッシュの処理負荷軽減も今後の課題です。
これらの技術を活かして今後のゲーム制作に取り組もうと思います。
